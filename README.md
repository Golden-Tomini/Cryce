## Getting Started

To get started with the Ingridentify Android App, follow these steps:

1. **Clone the Repository:**

   ```bash
   # SSH
   git clone git@github.com:Golden-Tomini/Cryce.git

   # HTTPS
   git clone https://github.com/Golden-Tomini/Cryce.git
   ```

2. **Run:**
   ```bash
    flutter run
   ```
